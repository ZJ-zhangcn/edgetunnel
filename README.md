# edgetunnel

CF Workers/Pages 边缘隧道（VLESS/Trojan 等）+ 管理面板与订阅。

| 项 | 值 |
|---|---|
| 仓库 | https://github.com/ZJ-zhangcn/edgetunnel |
| 上游 | https://github.com/cmliu/edgetunnel |

## 部署（Workers）

1. 将本仓库部署到 Cloudflare Workers / Pages  
2. 按上游教程配置环境变量与 KV/绑定  
3. 打开管理面板完成节点/订阅设置  

图文教程（上游）：https://cmliussss.com/p/edt2/

## 常用能力

- 协议：VLESS / Trojan / Shadowsocks 等  
- 订阅：Clash / Sing-box / Surge 等  
- 链式：ProxyIP、SOCKS5/HTTP  

细节与变量表以上游文档为准；本页只作 fork 入口。
